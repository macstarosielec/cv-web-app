# VS Code Configuration

The repository includes pre-configured VS Code settings:

- **`.vscode/settings.json`**: Points the Dart SDK to the FVM-managed Flutter installation (`.fvm/versions/stable`).
- **`.vscode/launch.json`**: Launch configurations for every app and package, including debug, profile, and release modes.

When opening the project in VS Code, the Dart extension will automatically pick up the FVM-managed SDK.
